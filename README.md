# Student Management System

A console-based Student Management System developed in **C++** that allows users to manage student records efficiently using file handling. The system provides a menu-driven interface to perform various operations such as adding, searching, updating, deleting, and displaying student information.

## Features

- Add new student records
- Display all student records
- Search student by ID
- Update existing student details
- Delete student records
- Persistent data storage using file handling
- Simple menu-driven console interface

## Technologies Used

- **Language:** C++
- **Concepts:** Classes, Functions, File Handling, Loops, Conditional Statements
- **IDE:** VS Code / CodeBlocks / Dev-C++
- **Compiler:** GCC (MinGW)
